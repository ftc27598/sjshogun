# SJ Shogun Website

Official website for SJ Shogun Team 27598.

## Getting Started

Requirements:
- Node.js 18+
- npm

Install and run:

```sh
npm install
npm run dev
```

Build for production:

```sh
npm run build
npm run preview
```

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
